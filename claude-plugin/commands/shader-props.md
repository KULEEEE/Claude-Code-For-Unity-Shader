---
name: shader-props
description: Get shader property definitions
---

Query a shader's property definitions using `get_shader_properties`.

Ask the user for the shader path if not provided as an argument.

Display the properties as a formatted table with:
- Property name
- Display name
- Type (Float, Color, Vector, Texture, etc.)
- Default value
- Attributes (HDR, NoScaleOffset, etc.)
